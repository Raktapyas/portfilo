# assets/

Place your images here and reference them in index.html.

## How to add your profile photo:

1. Add your photo to this folder (e.g. `profile.jpg`)
2. Open `index.html` and find the two `.photo-placeholder` divs
3. Replace each one with:
   <img src="assets/profile.jpg" alt="Basudev Dhakal" />

## Suggested assets to add:
- profile.jpg       → your profile/headshot photo
- favicon.ico       → browser tab icon
- og-image.png      → social media preview image (1200x630px)
- portfolio-1.jpg   → portfolio project screenshots
- portfolio-2.jpg
- portfolio-3.jpg
- ...
